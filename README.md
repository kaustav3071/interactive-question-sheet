# Interactive Question Sheet

A professional Striver SDE Sheet tracker built with React, Vite, and Tailwind CSS.

## Features
- Track questions by topic and difficulty
- Drag & drop reorder
- Add/edit/delete questions and topics
- Personal notes per question
- Progress stats and filters
- Export/import your sheet as JSON
- Persistent progress (localStorage)

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```
2. **Start dev server:**
   ```bash
   npm run dev
   ```
3. **Build for production:**
   ```bash
   npm run build
   ```
